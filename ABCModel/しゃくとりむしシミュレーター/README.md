# しゃくとりむしシミュレーター

**ABCモデルで虫の心理を可視化する、インタラクティブな行動シミュレーター**

## 概要

このアプリケーションは、尺取虫（しゃくとりむし）の独特な移動方法をABCモデルで表現します。

3つの心理的パラメータを調整することで、虫の歩行スピード、ためらい、震えがどのように変化するかを視覚的に体験できます。環境ストレスの効果も加わり、複雑な相互作用を観察できます。

## ABCモデルの各パラメータ

### A: 元気（Activation）
- **役割**：行動のエネルギー源
- **効果**：
  - 高い → 虫が素早く歩く、歩幅が大きくなる
  - 低い → 虫が歩みが遅い

### B: プレッシャー（Braking / 緊張度）
- **役割**：気持ちの引っかかりやストレス
- **効果**：
  - 高い → 虫が震える、細かく揺れる
  - 低い → なめらかな移動

### C: 考えすぎ（Constraint / メタ認知）
- **役割**：「今、自分は何をしているのか？」という思考の深さ
- **効果**：
  - 高い → 虫が長くためらう、行動が遅延
  - 低い → すぐに行動を開始

### E: 環境要因（Environment）
- **役割**：外部ストレス（例：雨、障害物）
- **効果**：
  - 高い → 空が暗くなり、雨が降り、虫のストレスが増加
  - 低い → 晴れた状態

## 虫の動きのサイクル

しゃくとりむしは3つのフェーズを循環します：

1. **THINK（思考）**：
   - 虫が立ち止まり、考える（Cが高いほどこの時間が長い）

2. **STRETCH（伸びる）**：
   - 前足でぐーっと前に伸びる（Aが高いほど素早く、歩幅も大きい）

3. **CONTRACT（縮む）**：
   - 後ろ足を前に引き寄せて、身体を丸める

## 使い方

### パラメータ調整

- **A（元気）スライダー**：0.0 ～ 2.0
- **B（プレッシャー）スライダー**：0.0 ～ 2.0
- **C（考えすぎ）スライダー**：0.0 ～ 2.0
- **E（環境ストレス）スライダー**：0.0 ～ 2.0

リアルタイムで虫の動きに反映されます。

### 状態表示

- **移動距離**：虫が実際に進んだ距離（相対値）
- **RuinScore**：総合的なストレス度（0～6）
- **各パラメータの状態**：Normal / Warning / Critical

## 心理学的な観察ポイント

🐛 **相互作用の複雑性**：
- A（元気）と B（プレッシャー）のバランスが大切
- 元気があればプレッシャーに強い
- 考えすぎが多いと元気がそがれる

🐛 **Zero状態（完全停止）**：
- 考えすぎ（C）が暴走し、かつ元気（A）が低い時に発生
- 虫がまったく動かなくなる

🐛 **Runaway状態（暴走）**：
- 元気またはプレッシャーが1.5以上になると暴走します
- バランスが崩れると危険

🐛 **環境のストレス効果**：
- 雨（高E）で虫の元気が奪われ、プレッシャーが増す
- 同時に思考が深くなり、ためらいが増える

## 学習目標

✨ **システムダイナミクス**：複数の変数が相互作用して生まれる行動パターン

✨ **心理学モデリング**：人間や動物の心理プロセスを数式で表現する方法

✨ **バランスの重要性**：一つのパラメータかけらだけが高ければよいわけではないこと

✨ **環境と個体の相互作用**：外部条件が内部状態にどう影響するか
