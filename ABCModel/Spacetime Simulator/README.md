# Spacetime Simulator

ABCモデルの心理力学を **相対性理論のメタファー** で可視化する、インタラクティブな時空シミュレータ。光・重力・観測という3つの要素が織りなす動的な宇宙を通じて、心の複雑な相互作用を直感的に理解します。

## 概要

従来の心理学の数値表示ではなく、アインシュタインの相対性理論の概念を用いて人間の感情力学を **物理的に表現** するユニークなシミュレータです。

本シミュレータでは、心の3つの軸が生み出す力学を時空の歪みとして可視化し、パラメータを調整することで、あなただけの内部宇宙を観察できます。

## 主要機能

### 3つのメタファー的パラメータ

#### A: Authenticity（光 / 固有時）
- **物理学的意味**: 光子の速度・エネルギー、軌道の活発さ
- **心理学的意味**: 自分自身のエネルギー、体感、自然な反応力
- **表示**: パーティクルの明るさ、速度、サイズ
- **スケール**: 0-2.0

#### B: Aesthetic（重力 / 歪み）
- **物理学的意味**: 時空を歪める重力場、中心への吸引力
- **心理学的意味**: 他者との比較、評価の基準、外部からの圧力
- **表示**: グリッドの収縮、中心星の大きさ、色の赤化
- **スケール**: 0-2.0

#### C: Meta（観測 / 境界）
- **物理学的意味**: 観測者の存在による事象の地平面、ビネット効果
- **心理学的意味**: 第三者の視点、監視されているという認知、自己検証
- **表示**: 外縁のビネット（紫ぼかし）、パーティクルの色相
- **スケール**: 0-2.0

### 物理シミュレーションエンジン

#### 1. 目標値への追従
- 各パラメータは目標値に指数近似で追従
- 速度係数: 0.05（円滑な変化）

#### 2. 変動量（デルタ）の計算
```
deltaA = min(2, |tA - cA| × 10)
deltaB = min(2, |tB - cB| × 10)
deltaC = min(2, |tC - cC| × 10)
```

#### 3. 状態判定
- **Runaway (1)**: v ≥ 1.6 → 暴走・崩壊状態
- **Normal (0)**: 0.2 < v < 1.6 → 安定状態
- **Zero (-1)**: v ≤ 0.2 → 凍結・停止状態

#### 4. 相互作用ロジック
相互作用係数 g(v) = 0.5 × tanh(0.7 × v)

```
calcA = cA + sgn(sC) × w(sC) × (-1) × g(cC) × boostC
calcB = cB + sgn(sA) × w(sA) × (-1) × g(cA) × boostA
calcC = cC + sgn(sB) × w(sB) × (+1) × g(cB) × boostB
```

**相互作用の意味**:
- **C→A (-1)**: 観測（監視）が強すぎると、光（エネルギー）を奪う
- **A→B (-1)**: 光（内なる力）がある程度あると、重力（外圧）を軽減する
- **B→C (+1)**: 重力（評価圧）が強いと、観測（自己監視）を強化する

#### 5. スパイク時のブースト
delta ≥ 1.5 の場合、相互作用を 1.35 倍に増幅（不安定性の増加）

#### 6. トランジションルール（TR）

| ルール | 条件 | 結果 | 意味 |
|--------|------|------|------|
| TR-1 | C暴走 ∧ ΔC Spike ∧ A弱い | A→Zero | 観測問題：時間凍結 |
| TR-2 | B暴走 ∧ A Zero | C→Runaway | 事象の地平面の完成 |
| TR-3 | A高揚 ∧ BのSwing | B→Normal | 内なる光による重力安定化 |

### ビジュアル要素

#### グリッド（空間の歪み）
- **calcB** に応じて中心に吸い込まれるような動き
- B暴走時は不規則な歪みが加わる
- 波のような周期変動で時間の流れを表現

#### 光子（パーティクル）
- **個数**: 150個のパーティクルが軌道を描く
- **速度**: calcA に比例
- **軌道半径**: calcB が大きいほど収縮
- **色**: 状態に応じて変色
  - **正常時**: 青系（冷静）
  - **A Zero時**: 灰色（凍結）
  - **B Runaway時**: 赤系（熱）
  - **C Runaway時**: 紫系（量子的不確定性）

#### 中心星（ブラックホール / 恒星）
- **サイズ**: 10 + calcB × 20 （B が大きいほど膨張）
- **色**:
  - **B Runaway**: 黒（ブラックホール、光さえ逃げられない）
  - **A Zero**: 灰色（死んだ星）
  - **正常時**: 青（恒星、光を放つ）
- **グラデーション**: 中心から外側へ放射状に色が薄れる

#### ビネット効果（観測の境界）
- **calcC > 0.1** で境界が現れる
- **C Normal**: 紫ぼやけ（自己意識の存在）
- **C Runaway**: 濃い赤紫、フリッカー（監視の過剰）

## 使用方法

### 基本操作

1. **パラメータ調整**: 下部のスライダーで A、B、C を 0~100 の間で動かす
   - 左端（0）= 最小値（0.0）
   - 右端（100）= 最大値（2.0）

2. **リアルタイム観察**: 
   - Canvas上が動的に変化
   - 左上のステータスが更新
   - パーティクル軌道、グリッド歪み、中心星が変形

3. **状態遷移を観察**:
   - 通常状態 → 高揚 → 暴走/凍結
   - 相互作用による現象を観察

4. **解説を読む**: 右上の ⓘ ボタンでモーダルを開く

### インタラクティブな実験

#### 実験1: 光だけを強める
- A: 100 → 200
- B: 0
- C: 0
→ パーティクルが活発に回転、明るく光る

#### 実験2: 重力だけを強める
- A: 50
- B: 100 → 200
- C: 0
→ グリッドが中心に吸い込まれ、中心星が膨張、重力崩壊の危機

#### 実験3: 観測だけを強める
- A: 50
- B: 0
- C: 100 → 200
→ ビネットが濃くなり、境界が現れる、パーティクルが微かになる

#### 実験4: B + C の暴走
- A: 50
- B: 180
- C: 180
→ TR-2ルール発動、事象の地平面の完成、紫色のビネットが激しくフリッカー

#### 実験5: A の内なる光で救済
- A: 180（高揚状態）
- B: 150（暴走状態）
- 観察 → calcB が低下、安定化

## 計算式詳細

### 状態判定関数
```javascript
getState(v):
  if (v >= 1.6) return 1;   // Runaway
  if (v <= 0.2) return -1;  // Zero
  return 0;                 // Normal
```

### 重み関数
```javascript
w(state):
  state === 1? 0.9 : (state === -1 ? 0.25 : 0)
  // Runaway は強く影響、Zero は弱く、Normal は中立
```

### 符号関数
```javascript
sgn(state):
  return state === 1 ? 1 : 0
  // Runaway なら効果あり、他は中立
```

### パラメータ構造

```javascript
// 内部シミュレーション状態
{
  tA, tB, tC:     // Target（ユーザー入力）
  cA, cB, cC:     // Current（追従中）
  calcA, calcB, calcC: // 相互作用後の最終値
  sA, sB, sC:     // State (-1/0/1)
  time:           // 経過時間（ms）
  particles: []   // 150個のパーティクルオブジェクト
}

// パーティクル
{
  angle:          // 現在の角度
  baseRadius:     // 基本軌道半径
  yOffset:        // Y方向のオフセット
  size:           // サイズ
  speed:          // 角速度
  hue:            // 色相（HSL）
}

// UI用ステータス
{
  A: { val: number, state: -1|0|1 },
  B: { val: number, state: -1|0|1 },
  C: { val: number, state: -1|0|1 },
  ruin: 0|2|3  // RuinScore簡易版
}
```

## 操作上のコツ

1. **段階的に変化**: 一度に複数パラメータを変えず、1つずつ観察

## 哲学的背景

このシミュレータは以下の仮説に基づいています：

> 「人間の心理力学は、マクロな物理法則（相対性理論）と同じ形式を持つ。光が曲がるように感情が曲がり、時間が止まるように心が止まる。」

相互作用ルールは中立ではなく、次のような哲学的事実を暗示しています：

- **観測は存在を変える**: 量子論の観測問題と同様、監視されると心は凍結する
- **内なる光がある時のみ圧力に抗える**: エネルギー（A）がなければ外圧（B）は制御不可
- **完全な暴走は不可逆的ではない**: TR-3のように、内なる光が回復を可能にする
