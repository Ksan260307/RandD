# Friendly Diagnosis App（フレンドリー ディアグノシス アプリ）

**12の自分らしさを、スライダーで触ってみる。** 複雑な性格分析を、感覚的でビジュアルにわかりやすく、友好的なUI で表現するインタラクティブ診断アプリです。

## 概要

このアプリは、ABCモデルを応用した **性格・適応パターン分析システム**を、むずかしい用語を避けて、誰でも直感的に操作できる形にまとめました。

プレイヤーは：
1. **12個のアクシス** （感性、抽象、集中、など）をスライダーで調整
2. 各アクシスの「出やすさ」「揺れやすさ」「外からの負荷」を3つの視点から入力
3. **レーダーチャート・バーチャート・カード UI** で、今の自分らしさ全体が見える化
4. 「1ステップ動かす」で時系列シミュレーション可能

という体験を通じて、自分のパターンを **客観的かつ視覚的に** 理解できます。

## 主要機能

### 1. 12個のアクシス（軸）

各アクシスは、性格の異なる側面を表現します：

| # | アクシス | 短縮名 | 絵文字 | 説明 |
|---|---------|--------|--------|------|
| 1 | 感じる力 | 感性 | ✨ | 刺激や気分の動きにどれくらい反応しやすいか |
| 2 | 考えをまとめる力 | 抽象 | 🧠 | 全体像をつかんだり、考えを整理したりする力 |
| 3 | 深く集中する力 | 集中 | 🎯 | ひとつのことに深く入り込む力 |
| 4 | 場になじむ力 | 適応 | 🤝 | 空気やルールに自然になじみやすいか |
| 5 | 人との距離感 | 距離感 | 🪐 | 人と近く関わるより、自分のペースを保ちやすいか |
| 6 | 気分のふり幅 | 感情幅 | 🌊 | 感情や行動の変化がどれくらい大きいか |
| 7 | 自分の基準の強さ | 自分軸 | 🧭 | 自分らしいこだわりや判断基準の強さ |
| 8 | 日々を回す安定感 | 安定感 | 🧱 | 予定や作業を着実に進めやすいか |
| 9 | ひらめきの出方 | 創造性 | 🎨 | 発想したり組み立てたりする力の出やすさ |
| 10 | 切り替えのしやすさ | 柔軟さ | 🍃 | 新しい情報や変化に合わせやすいか |
| 11 | 勢いの強さ | 勢い | ⚡ | 気持ちや刺激にすばやく動かされやすいか |
| 12 | 大事にしたい軸 | 価値観 | 💎 | 好き・正しさ・効率などの軸がどれくらいはっきりしているか |

### 2. 3つの診断パラメータ

各アクシスごとに、3つの側面から状態を測定します：

#### value（今の出やすさ）
- **範囲**: 0-100%
- **説明**: そのアクシスの力が、今どれくらい前に出やすいか
- **表現**: パーセンテージ + スター表記（■）
- **カテゴリ**:
  - 控えめ（< 35%）
  - ほどよい（35-55%）
  - はっきり（55-75%）
  - かなり強い（75%以上）

#### delta（最近の揺れやすさ）
- **範囲**: 0-100%
- **説明**: そのアクシスの値がどのくらい変動しやすいか、波が大きいか
- **カテゴリ**:
  - 落ち着きめ（< 15%）
  - ゆれあり（15-45%）
  - 大きく動きやすい（45%以上）

#### gate（外からの負荷）
- **範囲**: 0-100%
- **説明**: 環境や外部からのストレス・圧力がどれくらいかかっているか
- **影響**: 高いほど、アクシスの normal/runaway/zero 状態が固定化しやすくなる

### 3. 3つの状態（State）

各アクシスの現在の心理状態を示します：

| 状態 | 色 | 説明 | 表現 |
|------|------|------|------|
| **Normal（安定）** | 🟢 緑 | いつもの出方に近い状態 | 「安定」 |
| **Runaway（高ぶり気味）** | 🟠 オレンジ | 勢いが出すぎて空回りしやすい状態 | 「高ぶり気味」 |
| **Zero（閉じ気味）** | 🔵 青 | 力が内側にこもって出にくい状態 | 「閉じ気味」 |

状態遷移のロジック：
```
zeroTriggered = (value ≤ 25%) + (delta < 10%) + (quiet history) + (gate ≥ 70%) ≥ 2
  ↓
holdZero = zeroLock > 0 OR (prevState === "Zero" AND value < 35%)
  ↓
triggerRunaway = (value ≥ 80% AND delta ≥ 45%) OR (gate ≥ 70%)
  ↓
currentState = holdZero ? "Zero" 
            : (holdRunaway OR triggerRunaway) AND zeroLock === 0 ? "Runaway" 
            : "Normal"
```

### 4. 5つの手応え感（Dimension）

12個のアクシスから、5つの重要なディメンション（手応え感）を計算します：

| ディメンション | 構成 | 説明 |
|-------------|------|------|
| **ひらめき** | (感性 + 抽象 + 創造性) / 3 | 創造的な活動に向いているか |
| **深掘り** | (集中 + 自分軸 + 抽象) / 3 | ひとつのことを深くできるか |
| **対人バランス** | (適応 + (1-距離感) + 柔軟さ) / 3 | 人付き合いのしやすさ |
| **安定運転** | (安定感 + 柔軟さ + (1-勢い)) / 3 | 日常を着実に進められるか |
| **感情の動き** | (感情幅 + 勢い + 感性) / 3 | 感情表現の豊かさ |

各ディメンションが**トップ3**として表示され、その人の特徴が可視化されます。

### 5. 6つの性格パターン

12個のアクシスと5つのディメンションから、その日の性格パターンを自動判定：

| パターン | 条件 | 説明 | 相性よい人 |
|---------|------|------|----------|
| **探究型** | creative ≥ 0.72 & deepWork ≥ 0.68 | 感性と深掘りが両立 | 質問を重ねてくれる人 |
| **実務型** | steady ≥ 0.72 & social ≥ 0.62 | 安定と対人バランスが高い | 実行をサポートしてくれる人 |
| **独立型** | 距離感 ≥ 0.75 & 自分軸 ≥ 0.75 | 自分のペースと基準が強い | 自分を尊重してくれる人 |
| **躍動型** | 感情幅 ≥ 0.72 | 気分の勢いが表現に出やすい | 感情をキャッチしてくれる人 |
| **バランス型** | その他 | 複数の側面を行き来する | 臨機応変な人 |

その日の「気候」も評価：
- 全体が安定：「全体は比較的安定しています」
- Runaway ≥ 3：「熱量が高めです」
- Zero ≥ 3：「少し負荷がかかっています」
- Runaway ≥ 2 & Zero ≥ 2：「波の大きい状態です」

### 6. 計算ロジック（複雑な状態遷移）

#### ステップ1：状態判定
各アクシスについて、内部スコアに基づいて state を判定。

#### ステップ2：delta（揺れ）の計算
```
nextDelta = delta
if (value >= 0.75 && delta >= 0.15) nextDelta = max(delta - 0.15, 0)
if (!previousHadRunaway) nextDelta = max(delta - 0.2, 0)
```
高い値でまとまっているほど、揺れが減少。system全体で Runaway がなくなると、すべてのdelta が減少。

#### ステップ3：zeroLock（固定化）の計算
```
nextZeroStreak = zeroTriggered ? zeroStreak + 1 : 0
if (nextZeroStreak >= 2 || (gate > 0.8 && zeroTriggered)) {
  nextZeroLock = 2  // 2ステップ間ロック
} else if (!previousHadRunaway) {
  nextZeroLock = max(zeroLock - 1, 0)  // 回復
}
```

#### ステップ4：value（出やすさ）の更新
```
if (currentState === "Runaway" && nextZeroLock === 0) {
  nextValue = clamp(value + nextDelta × (1 - gate) × trend)  // 加速
} else if (currentState === "Zero") {
  nextValue = clamp(value - max(0.05, gate × 0.08 + nextDelta × 0.35))  // 低下
} else {
  nextValue = clamp(value + trend × nextDelta × 0.18 × (1 - gate))  // 緩和
}
```

### 7. 5つのプリセット

クイック診断のためのプリセット：

| プリセット | 説明 | 典型的なシナリオ |
|-----------|------|----------|
| **バランス型** | 全体をまんべんなく見たいとき | 普通の日 |
| **ひらめき重視** | 感性・発想・集中が強め | クリエイティブな作業モード |
| **堅実型** | 安定感と実行力が高め | ルーティンをこなす日 |
| **マイペース型** | 自分の基準と距離感がはっきり | 一人の時間を重視したい日 |
| **揺れやすい日** | 外からの負荷が大きい状態 | ストレスが多い状態 |

### 8. インタラクティブ機能

#### 1ステップ動かす
全12個のアクシスを同時に、複雑な相互作用ロジックに従って 1ステップ進める。ステップ数で時系列シミュレーション可能。

#### 少し落ち着かせる
すべてのアクシスの delta と gate を減少させ、Runaway状態を Normal に戻す。「心を落ち着ける」ボタン。

#### 詳細調整を開く
各アクシスの delta（揺れやすさ）と gate（外からの負荷）を個別に調整可能なエクスパンドパネル。

#### 初期状態へ
すべてをデフォルト値にリセット。

## 技術スタック

- **フレームワーク**: React 18+（TypeScript）
- **アニメーション**: Framer Motion
- **チャート**: Recharts（RadarChart, BarChart）
- **スタイリング**: Tailwind CSS
- **アイコン**: 絵文字（デザイン一貫性）
- **状態管理**: React Hooks (useState, useMemo, useEffect)
- **永続化**: localStorage（state 自動保存）
- **計算**: 複雑な非線形フィードバック・条件分岐

## 使用方法

### 基本的な流れ

1. **プリセット選択（オプション）**
   - 上二段の5つのプリセットから近いものをクリック
   - または自分で調整開始

2. **アクシス調整**
   - 左側スライダーで「今の出やすさ」を調整（最重要）
   - 各アクシスの下に「出方」「揺れ」「ひとこと」がリアルタイム表示

3. **ダッシュボード確認**
   - 右側に「性格パターン」「Climate」「Dimension ランキング」が自動計算
   - レーダーチャートで全体バランスを可視化
   - バーチャートで「強さと揺れの見比べ」

4. **詳細調整（必要に応じて）**
   - 各カードの「くわしく」をクリック
   - delta（揺れやすさ）と gate（外からの負荷）を個別調整
   - trend（向き）を上/下に選択

5. **シミュレーション**
   - 「1ステップ動かす」で次のシナリオを見る
   - 複数回実行して、状態がどう変化するか観察

#### 実験シナリオ

##### 実験1：「ストレスが増えると、何が起きるか」
```
初期状態（バランス型）から：
1. 感性 + 感情幅の gate を 70% に
2. 「少し落ち着かせる」で値を下げる
3. 「1ステップ動かす」を3~5回実行

期待結果:
- 感性と感情幅が Runaway → Zero に遷移する可能性
- 気分のふり幅が大きく変化
- climate が「波の大きい状態」に
```

##### 実験2：「ひらめきモード→堅実モードの移行」
```
1. ひらめき重視 プリセット をクリック
   → 感性、抽象、創造性が 75-90% に
2. 「1ステップ動かす」を 10回実行
3. 感性の delta を 0% に落とす
4. 安定感・適応の value を上げる

期待結果:
- 徐々に「探究型」 → 「バランス型」に遷移
- ひらめきスコアが低下、安定運転スコアが上昇
```

##### 実験3：「独立型の人が、対人ストレスを受けたら」
```
1. マイペース型 プリセット（距離感・自分軸高め）
2. 社会圧を表現するため：
   - 適応の gate を 80% に
   - 感情幅の delta を 60% に
3. 「1ステップ動かす」を 5回実行

期待結果:
- 適応（社会適応）が Runaway に
- 距離感に conflicting pressure
- climate が「高ぶり気味」に
```

##### 実験4：「揺れやすい日の復帰プロセス」
```
1. 揺れやすい日 プリセット
   → すべて delta 高め、gate 70%
2. 「少し落ち着かせる」を 3回クリック
3. 「1ステップ動かす」を 5回実行

期待結果:
- delta と gate が段階的に低下
- 多くの state が Zero → Normal に復帰
- climate が「落ち着きあるバランス」へ
```

## ビジュアルデザイン

### レイアウト構成
- **上段**: ヘッダー（ロゴ、タイトル、操作ボタン）、5つのプリセット
- **主パネル**: 2分割
  - **左（main）**: 12個のアクシスカード（2列グリッド）
  - **右（sidebar）**: 3つの情報パネル
    1. 「全体ひとこと」 + climate
    2. 「全体のひとこと」パネル（state distribution）
    3. 「全体バランス」レーダーチャート
    4. 「強さと揺れの見比べ」バーチャート
    5. 「読み取りのヒント」ガイドパネル

### 色彩システム
- **Normal**: 緑（bg-emerald-50）
- **Runaway**: 橙（bg-amber-50）
- **Zero**: 青（bg-sky-50）
- **背景**: グラデーション（白→緑→青の優しいトーン）
- **Dimension**: violet-700（上位3つをハイライト）

### インタラクティブ要素
- **スライダー**: RGB グラデーション背景
- **ラジオ/ボタン**: active で bg-slate-900, inactive で border
- **カード**: hover で shadow 増加、layout animation
- **グラフ**: ResponsiveContainer で自動スケール

## 心理学的背景

このシステムは以下の概念を統合しています：

### Big Five と性格心理学
- 感性、適応、距離感 → Openness / Agreeableness
- 安定感、勢い → Conscientiousness / Extraversion
- 感情幅 → Neuroticism

### 複雑系・ダイナミカルシステム
- state transition は確率的ではなく決定論的（reproducible）
- 過去の状態（prevState）が現在に影響（path-dependent）
- 複数パラメータの相互作用（delta × gate × value）

### ストレス反応の 3パターン
- Runaway（過剰適応）
- Normal（適応的）
- Zero（無反応）

### HSP（高刺激処理感受性）と個人差
- gate が高い人ほど、少しの変化で state 遷移しやすい
- delta が高い人は波が大きく、意思決定が変わりやすい

## UI/UX 設計原則

1. **友好的**: むずかしい用語を避け、絵文字と短い言葉で表現
2. **視覚的**: レーダー・バーチャートで全体の形を即座に把握可能
3. **操作的**: スライダーで直感的に状態変化を感じられる
4. **段階的**: 基本3つの入力 → 詳細3つの拡張 → シミュレーション
5. **リアルタイム**: 入力するたびに全体が即座に更新

## 応用例

- **自己認識**: 今の自分のバランスを客観的に把握
- **ストレスマネジメント**: 負荷を可視化して対策を立てる
- **チームビルディング**: メンバーの特性を理解して配置
- **心理学教育**: 複雑なシステムの相互作用を直感的に学ぶ
- **創造性支援**: 自分のひらめきモードと堅実モードの切り替えを意識的に
- **コーチング**: クライアントの状態変化を可視化して会話のきっかけに

## 注記

- このアプリは **自己認識ツール** であり、医学的診断ではありません
- アクシスの値・状態は **相対的な参考**です（絶対値ではない）
- 実際の人間の複雑さはさらに大きく、個人差・環境差に左右されます
- localStorage に状態が自動保存されるため、ブラウザを閉じても復帰可能
