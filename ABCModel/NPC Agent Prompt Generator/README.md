# NPC Agent Creator

NPC の性格・心理状態をパラメータで設計し、  
その設定をもとに **AI 用プロンプト生成** と **セリフサンプル生成** を行う React アプリです。

## 概要

`NPC Agent Creator` は、キャラクターの心理を以下の 3 レイヤーで扱います。

- **A: 心・感情**
- **B: 理想・信念**
- **C: 周囲・客観性**

それぞれの強さ・不安定さ・状態を調整することで、  
NPC の精神状態を可視化しながら、AI に渡すためのプロンプトを自動生成できます。

さらに、現在の設定をもとに AI から **セリフサンプル** を生成し、  
キャラクターの雰囲気をその場で確認できます。:contentReference[oaicite:1]{index=1}

## 主な機能

### 1. 基本情報の設定
以下の基本プロフィールを入力できます。

- キャラクター名
- 役割・職業
- 性格・口調

### 2. ABC モデルによる心理設計
3 つの心理レイヤーを個別に調整できます。

#### A: 心・感情
- 感情の強さ
- 不安定さ
- 状態（平常心 / 暴走中 / 無反応）

#### B: 理想・こだわり
- こだわりの強さ
- 迷い
- 状態
- タイプ  
  - 自分に厳しい（内向き）
  - 人の目が気になる（外向き）

#### C: 周囲への意識
- 気にする度
- 疑心暗鬼
- 状態
- タイプ  
  - 論理的
  - 空気を読む
  - 自意識過剰
  - 常識人

### 3. 精神負荷度の可視化
各レイヤーの値と状態をもとに、  
NPC の **Mental Stability** を百分率で表示します。

- Stable
- Unstable
- Critical

### 4. AI 用プロンプト生成
設定した内容をまとめて、AI にそのまま渡せるプロンプトを生成します。

### 5. セリフサンプル生成
Gemini API を使い、現在のキャラクター設定に応じた  
短いセリフを 3 つ JSON 配列形式で取得します。

### 6. ランダム設定
各心理パラメータをランダムに変更して、  
偶発的で面白いキャラクターを素早く作れます。:contentReference[oaicite:2]{index=2}

## セットアップ

### 1. リポジトリを取得

```bash
git clone <your-repo-url>
cd <your-project-name>
````

### 2. 依存関係をインストール

```bash
npm install
```

### 3. API キーを設定

コード内では現在、以下のように API キーを直接変数に入れる構成です。

```js
const apiKey = "";
```

ここに Gemini API キーを設定してください。

ただし、実運用では **環境変数化** を強く推奨します。

例:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

または

```env
REACT_APP_GEMINI_API_KEY=your_api_key_here
```

利用しているビルド環境に合わせて参照方法を変更してください。

### 4. 開発サーバーを起動

```bash
npm run dev
```

または

```bash
npm start
```

※ 実行コマンドは使用している環境（Vite / CRA など）に合わせてください。

## Gemini API について

このアプリはセリフ生成時に、以下のような流れで API を呼び出します。

* system prompt で「ゲームのシナリオライター」として振る舞わせる
* user prompt で NPC の現在設定を渡す
* JSON 配列形式でセリフを 3 つ返させる

現在は次のモデル名を利用しています。

```txt
gemini-2.5-flash-preview-09-2025
```

必要に応じて、利用可能なモデル名へ変更してください。

## 使い方

1. キャラクター名・役割・性格を入力
2. A / B / C の心理パラメータを調整
3. 補足指示で追加設定を書く
4. 右側プレビューで精神負荷度を確認
5. セリフ生成ボタンでサンプルを確認
6. プロンプトをコピーして他の AI ワークフローに活用
