# 木を探索するココロの動き：二分探索木×ABCモデル可視化

**アルゴリズムの動きを心理的な視点から体感するインタラクティブシミュレーター**

## 概要

このアプリケーションは、二分探索木（Binary Search Tree）のアルゴリズムを、ABCモデル（心理学的行動制御モデル）と組み合わせて可視化します。

3つの心理的要素を調整することで、木の探索パフォーマンスがどのように変化するかを観察できます。

## 特徴

- **リアルタイム可視化**：二分探索木の構造と探索ルートを動的に表示
- **ABCパラメータ調整**：3つの心理的要素を独立して操作
- **状態監視**：負荷スコア（RuinScore）で探索の健全性を追跡
- **インタラクティブ操作**：スライダーで瞬時にシミュレーションに反映

## ABCモデルとは

### A：勢い（Activation / 直感力）
- **役割**：探索を前に進めるエネルギー
- **高すぎると**：無謀な選択、計算暴走（Runaway）
- **低すぎると**：探索の停止、意思決定麻痺（Zero）

### B：審判（Braking / 比較力）
- **役割**：左右の選択を厳密に比較するコスト
- **高すぎると**：過度な慎重さ、判断麻痺
- **低すぎると**：軽率な選択、エラー増加

### C：視点（Constraint / 俯瞰力）
- **役割**：全体の状況を見守り、行き過ぎを防ぐ
- **高い**：バランス型探索、エラー回避
- **低い**：局所最適、全体の見落とし

## 使い方

### 基本操作

1. **スライダー調整**
   - A（勢い）、B（審判）、C（視点）をそれぞれ 0 ～ 2.0 の範囲で設定
   - リアルタイムで可視化に反映されます

2. **ターゲットを探索**
   - 「ターゲットを探索」ボタンをクリック
   - 木の中から目標値を探すシミュレーションが開始
   - 探索ルートはハイライトされます

3. **木を再生成**
   - 新しい二分探索木を生成
   - ノード数やバランスがランダムに変わります

### 状態表示

- **RuinScore**：探索の負荷度合を 0% ～ 100% で表示
  - 低い：安定した探索
  - 高い：計算負荷が危険水準（Runaway状態に注意）

- **ステータスバッジ**：各パラメータの現在の状態
  - Normal：最適範囲
  - Warning：要注意
  - Critical：危険状態
