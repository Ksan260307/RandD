# 時空観測シミュレーター

このプロジェクトは、TVD（Triadic Velocity Dynamics）理論を応用した「時空観測シミュレーター」です。

## 概要
- 時空（空間と時間）のダイナミクスを、TVD理論に基づいてシミュレート・可視化します。
- TVD理論の教育・研究用途や、物理現象の直感的理解を目的としています。

## ファイル構成
- `main.html` : シミュレーター本体のHTMLファイル

## 使い方
1. `main.html` をブラウザで開いてください。
2. シミュレーションが自動的に開始されます。
3. 時空の動きや変化を観察できます。

## 必要環境
- モダンなWebブラウザ（Google Chrome, Firefox, Edgeなど）
