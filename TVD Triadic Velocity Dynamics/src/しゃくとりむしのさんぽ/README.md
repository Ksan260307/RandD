# しゃくとりむしのさんぽ

**TVDモデルで虫の心と体を動かす、リアルタイム相互作用シミュレーター**

## 概要

このアプリケーションは、「時間変動ダイナミクス（TVD: Time-variant Dynamics）」という動的システムモデルを用いて、しゃくとりむしの行動を精密にシミュレートします。

虫の内部状態（拡張・収縮・エネルギー）を可視化し、環境変化やユーザーの介入で虫がどのように心身を反応させるかを体験できます。

## TVDモデルの3つの状態変数

### D: のびる力（Drive / 拡張エネルギー）
- **役割**：前に進もうとする力、探索への推進力
- **高い状態**：虫が伸びている、興奮している
- **低い状態**：おとなしい、静かに収縮中

### R: ちぢむ力（Resilience / 収縮力）
- **役割**：身体を閉じる力、内向的な状態
- **高い状態**：虫が丸まっている、引っ込まれている
- **低い状態**：身体が伸びきっている

### F_int: げんき（Internal Field / 内部場）
- **役割**：全体的なエネルギー基盤、心身の活力
- **高い**：元気一杯、活発に行動
- **低い**：疲れている、眠たそう

## 虫の動きのメカニズム

### 正常な歩行サイクル
1. **D が R より大きい**：虫が伸びる（前足を前に出す）
2. **R が D より大きい**：虫が縮む（後ろ足を前に寄せる）
3. **この繰り返し**：自然な尺取虫の歩行が生まれる

### 相互作用ルール
- **D主導** → より速く、より大きく伸びる
- **R主導** → より速く、より完全に閉じる
- **(D - R)** → 虫が前に進む力の指標

## ユーザー操作

### パラメータ調整

#### ☀️ まわりの環境（F_ext）
- スライダー：0.0 ～ 1.0
- **高い（0.5以上）**：天気が良い、虫が活発になる、背景が明るくなる
- **低い（0.5未満）**：天気が悪い（雨）、虫が内向的になる、画面が暗くなる

#### 👉 ちょっかい（Input）
- スライダー：-1.0 ～ +1.0
- **プラス（押す）**：虫を前に押す、促進
- **マイナス（驚かせる）**：虫を後ろに押す、ビックリ表情に
- **0（何もしない）**：自然な歩行を観察

#### 🏃 マイペースさ（lambda_eff）
- 範囲：0.05 ～ 0.5
- **高い**：行動に軽さがある、すばやく反応する虫
- **低い**：もったり振る舞う、粘り強い虫

#### 😴 つかれやすさ（epsilon）
- 範囲：0 ～ 0.1
- **高い**：虫がすぐに疲れ果てる、エネルギーを失うのが早い
- **低い**：タフな虫、持久力がある

### 虫への直接介入

三角形のマーカーを **ドラッグ** して虫を直接押すこともできます：
- **右にドラッグ**：虫を前に押す
- **左にドラッグ**：虫を後ろに押す、ビックリさせる

## 虫の表情と心理状態

虫の目の表現が、内部状態を反映します。

### 😊 通常の表情
- **条件**：F_int が中程度、落ち着いた状態
- **見た目**：黒い瞳がキラキラ

### 😴 眠たい表情
- **条件**：F_int < 0.3（元気がない）
- **見た目**：目が一本線になっている

### 😲 驚いた表情
- **条件**：Input < -0.1（驚かされた）
- **見た目**：×のような目

### 😆 興奮した表情
- **条件**：D > 0.8（のびる力が強い）
- **見た目**：赤いほっぺた

## 内部状態メーター

画面上部の **「いまの内部状態」** セクションで、リアルタイムに以下を確認できます：

- **🔴 のびる力（D）**：伸び出したい欲動
- **🔵 ちぢむ力（R）**：身を守りたい傾向
- **💛 げんき（F_int）**：全体的な活力レベル
- **💫 すすむ力（D-R）**：前進のネット力
- **⚡ かつどう量**：3つの要素の合計活動度

## 学習と実験のアイデア

🔬 **環境の影響を観察**：
天気を変えるとどのように虫の内部状態が変わるか試そう

🔬 **自分のペースを発見**：
「マイペースさ」を変えることで、虫のキャラが変わる。どの設定で虫が最も生き生きしているか？

🔬 **ストレスレジリエンス**：
つかれやすさを高くしてから何度も虫を押すと、虫がどのように疲れるか観察

🔬 **最適な歩行を探す**：
D と R のバランスがどの時に最も安定した歩行になるか

🔬 **表情の変化**：
各パラメータを極端に変えて、虫の表情がどう変わるか見よう

🔬 **カオスと秩序**：
lambda_eff を非常に低くすると、虫の装置が暴走するか、または安定するか？

## 理論的背景

TVDモデルは、複雑な生物学的・心理的な行動パターンを、シンプルな微分方程式で説明します。

**基本方程式の概略**：
- dD/dt = G_D × (F_total - R) - λ × D(1-D) - ε × D² + α × tanh(Input)
- dR/dt = G_R × (D - F_total) - λ × R(1-R) - ε × R²
- dF/dt = G_F × (R - D) - λ × F(1-F) - ε × F²

ここで：
- G_*（行動ゲイン）：シグモイド曲線で計算される非線形増幅
- F_total = F_int + F_ext：内部・外部環境の統合
- λ（マイペースさ）：減衰・運動の柔軟性を制御
- ε（つかれやすさ）：自己減衰、疲労の蓄積
