# 👻 Ghost VPN 超やさしい使い方ガイド

こんにちは！この説明書は、「Ghost VPN（ゴースト・ブイピーエヌ）」の使い方を、専門用語を使わずに世界一わかりやすく解説するガイドです。

## 🌟 Ghost VPNってなに？

Ghost VPNは、**「あなたの通信を、透明な幽霊のようにして守る」** 魔法のプログラムです。

インターネットの世界には、悪いハッカー（通信を盗み見ようとする人）がたくさんいます。Ghost VPNを使うと、あなたの通信が「カチカチの暗号カプセル」に包まれ、ダミーデータでカモフラージュされるため、誰にも中身を見られず、邪魔されることもなくなります！

### 🐒 サルでもわかる！通信のイメージ

**❌ 普通の通信（あぶない）**
💻あなた ――(丸見えのハガキ)――> 🕵️悪者（ニヤニヤ） ――> 🌍インターネット

**✨ Ghost VPNの通信（あんしん！）**
💻あなた ――(絶対に開かない魔法の金庫)――> 👻Ghostサーバー ――> 🌍インターネット
*※悪者には「意味不明なゴミのカタマリ」にしか見えません！*

---

## 🛠️ 準備するもの

* **パソコン** (WindowsでもMacでもOK！)
* **Python (パイソン)** というソフトがパソコンに入っていること。
* ダウンロードした `secure_tunnel_vpn.py` というファイル。

---

## 🚀 さっそく使ってみよう！（基本のつなぎ方）

Ghost VPNには「サーバー（受け取り係）」**と**「クライアント（送り係）」の2つの役割があります。この2つをつないで、秘密のトンネルを作ります。

### 🐾 ステップ1：サーバー（受け取り係）を起動しよう！

まずは、待っていてくれる「サーバー」を動かします。

1. パソコンで「ターミナル（黒い画面）」を開きます。
2. `secure_tunnel_vpn.py` がある場所で、次のコマンドを入れて **Enterキー** を押します！
```bash
python secure_tunnel_vpn.py

```


3. 画面の質問に答えます。（よくわからなければ、すべてそのままEnterでOK！）
* **起動モード:** `server` と入力してEnter！
* **秘密鍵:** お好きな合言葉（パスワード）を入れてEnter！（例: `my_secret_123`）
* その他の設定も、基本はそのままEnterを押していけばOKです。



👉 **「✅ [設定完了] Serverモードで起動します」** と出たら準備完了！
さらに画面の下に **「🔗 【クライアント側への接続案内】」** として、**接続用のコマンド（魔法の呪文）** が表示されるので、それをコピーしておきましょう！

---

### 🐾 ステップ2：クライアント（送り係）を起動しよう！

次に、あなたの手元で動く「クライアント」を動かします。別のパソコンでも、同じパソコンの新しい黒い画面でもOKです。

1. ステップ1でコピーした **「💻 接続コマンド」** を、黒い画面にそのまま貼り付けて（ペーストして）Enterを押すだけ！
*(コマンドの例)*
`python secure_tunnel_vpn.py --mode client --server-host 192.168.x.x --server-port 9999 --key my_secret_123`

👉 **「✅ [設定完了] Clientモードで起動します」** と出たら大成功！
たったこれだけで、**「誰にも見えない秘密のトンネル」** が完成しました！🎉

---

### 🐾 ステップ3：次回からはもっと簡単！（自動保存機能）

一度設定を完了すると、パソコンがあなたの設定を `ghost_vpn_config.json` というファイルに自動でメモしてくれます。

次回からはコマンドを打つと……
👉 **「📁 前回保存された設定が見つかりました。」**
👉 **「👉 前回の設定で起動しますか？ (Y/n)」**

と聞かれるので、**そのままEnterを押すだけ**で、一瞬でいつものトンネルが起動します！ラクチンですね！

---

## 🛡️ おまけ：テストモードでハッカーと戦ってみよう！

Ghost VPNがどれくらいスゴイのか、ハッカーの攻撃をシミュレーションする「テストモード」で遊んでみましょう！

1. 新しくプログラムを起動します。
```bash
python secure_tunnel_vpn.py

```


2. 「前回の設定で起動しますか？」と聞かれたら `n` を押してEnter。
3. 起動モードで `test` と入力してEnter！

**ドバババッ！と画面に文字が流れます！**
ここではGhost VPNに対して「データを書き換える」「偽物のハンコを押す」「通信を途中でぶった切る」といったサイバー攻撃を**1000回**連続で仕掛けています。

👉 **「🛡️ 総合防御成功率 : 100.00 %」**
どんなに攻撃されても、Ghost VPNがしっかりガードしているのがわかります😎

## 🌍 離れた場所（インターネット越し）で使うには？

家と会社など、離れた場所で繋ぎたい時は、**「サーバー側のWi-Fiルーター」** で少しだけ設定が必要です。

1. サーバーがある家のWi-Fiルーターの設定画面を開きます。
2. **「ポート開放（ポートフォワーディング）」** というメニューで、「外から `9999` 番に来た通信を、このパソコンに通す」という設定をします。
3. クライアント側で繋ぐときのIPアドレスを、サーバー側の「グローバルIPアドレス（インターネット上の住所）」に変更すれば完了です！
