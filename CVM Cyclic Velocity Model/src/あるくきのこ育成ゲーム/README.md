# あるくきのこ育成ゲーム

「あるくきのこ育成ゲーム」は、CVM（Cyclic Velocity Model）理論を応用した、歩くきのこの成長と行動を楽しむシミュレーションゲームです。

## 特徴
- きのこが自律的に歩き回り、成長・進化していく様子を観察できます。
- プレイヤーは環境や成長条件を調整し、きのこの行動や成長パターンに影響を与えられます。
- CVM理論を直感的に体験できる教育・研究用途にも最適です。

## 使い方
1. React環境で本プロジェクトをビルドしてください。
2. `main.jsx` をReactプロジェクトに組み込むか、ビルド後のアプリをブラウザで開きます。
3. 画面上の指示に従い、きのこの成長や行動を観察・操作してください。
4. 環境パラメータや成長条件を変更して、さまざまなシミュレーションを試せます。

## ファイル構成
- `main.jsx` : ゲーム本体のReactコンポーネント

## 必要環境
- Node.js および npm
- React 17以降
- モダンなWebブラウザ（Google Chrome, Firefox, Edgeなど）
