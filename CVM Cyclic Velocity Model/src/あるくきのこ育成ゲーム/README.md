# あるくきのこ育成ゲーム

このプロジェクトは、CVM（Cyclic Velocity Model）理論を応用した「あるくきのこ」の育成シミュレーターです。

## 概要
- きのこが歩く様子や成長過程を、周期的な速度モデル（CVM）に基づいてシミュレート・可視化します。
- CVM理論の教育・研究用途や、生物・物理現象の直感的理解、ゲーム的な楽しさを目的としています。

## ファイル構成
- `main.jsx` : ゲーム本体のReactコンポーネント

## 使い方
1. 必要に応じてReact環境でビルドしてください。
2. `main.jsx` をReactプロジェクトに組み込むか、ビルド後のアプリをブラウザで開いてください。
3. 画面上で「あるくきのこ」の成長や動きを観察・操作できます。

## 必要環境
- Node.js および npm
- React 17以降
- モダンなWebブラウザ（Google Chrome, Firefox, Edgeなど）
