視界による実体化デモ

マウス（または指）の周囲だけが「観測領域」です。
見ていない場所は「ぼんやりした雲」として計算を省略し、見た瞬間だけ中身が「個別の実体」として動き出します。これにより極限まで計算負荷を下げます。

総オブジェクト数 0
現在計算している実体数 0
計算を省略できた割合 0%

コアロジック（疑似コード） 

// 毎フレーム実行される更新処理
function updateSystem() {
  for (let cloud of clouds) {
    // 視界（カメラ/マウス）との距離を計算
    let dist = getDistance(cloud.x, cloud.y, camera.x, camera.y);

    if (dist > viewRadius) {
      // 【確率雲への圧縮】
      // 視界の外なら、詳細な計算をスキップして「雲」としてまとめて描画
      drawCloud(cloud);
      continue; // ← ここで処理を打ち切るのが超軽量化の鍵！
    }

    // 【観測による実体化】
    // 視界に入った雲の中身（何百もの個体）を展開し、それぞれ計算する
    for (let i = 0; i < cloud.particleCount; i++) {
      let particle = cloud.particles[i];
      
      // 重い物理演算や状態の更新を実行
      particle.updatePhysics();
      
      // 個別の実体として描画
      drawParticle(particle);
    }
  }
}
※ 実際の設計では、これらをGPU（Compute Shader）や固定長配列（Zero-Allocation）を用いてさらに極限まで高速化しています。
