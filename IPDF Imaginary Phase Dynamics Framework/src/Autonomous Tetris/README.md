# Autonomous Tetris

このプロジェクトは、IPDF（Imaginary Phase Dynamics Framework）理論を応用した自律型テトリスシミュレーターです。

## 概要
- テトリスの自動プレイアルゴリズムを実装し、ブロックの落下や回転、ライン消去などの挙動を自律的に制御します。
- IPDF理論の研究・教育用途や、AIアルゴリズムの可視化・検証を目的としています。

## ファイル構成
- `main.html` : シミュレーター本体のHTMLファイル

## 使い方
1. `main.html` をブラウザで開いてください。
2. テトリスの自動プレイが開始されます。
3. ブロックの動きやAIの判断を観察できます。

## 必要環境
- モダンなWebブラウザ（Google Chrome, Firefox, Edgeなど）
