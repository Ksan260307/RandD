# 未来予知アプリ「星導の羅針盤」

このプロジェクトは、IPDF（Imaginary Phase Dynamics Framework）理論を応用し、未来予知や意思決定支援をテーマにしたWebアプリケーションです。

## 概要
- ユーザーの入力や選択に基づき、未来の可能性や選択肢を可視化します。
- 星導（せいどう）の羅針盤をモチーフに、直感的なインターフェースで未来予測を体験できます。
- 教育・研究用途や、自己分析・意思決定のサポートを目的としています。

## ファイル構成
- `main.jsx` : アプリ本体のReactコンポーネント

## 使い方
1. 必要に応じてReact環境でビルドしてください。
2. `main.jsx` をReactプロジェクトに組み込むか、ビルド後のアプリをブラウザで開いてください。
3. 画面の指示に従い、未来予知や意思決定支援を体験できます。

## 必要環境
- Node.js および npm
- React 17以降
- モダンなWebブラウザ（Google Chrome, Firefox, Edgeなど）
